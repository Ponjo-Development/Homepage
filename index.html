PonjoAPI

The Ponjo API is an easy-to-use RESTFUL API for developers, with astonishingly simple endpoints. Developing, debugging, and/or testing your web or mobile application can be quite challenging to do without having any sort of data to work with. This is the main reason for which the Ponjo API exists! It is a simple API that allows beginner developers to work with aforementioned data to test their own applications. Use any of our API endpoints to fetch data to make testing your web application a breeze. Created and developed by Eerie, the API has countless features, some of which are listed below.

  • Utility Endpoints
  • Game Endpoints
  • Fun/Entertainment Endpoints

Developers can easily pass in a valid route and get a multitude of schemes. The API has a wide variety of modules, with complete documentation that is easily understandable for developers of all skill levels.

The Helios Network

Helios Practice was a public Minecraft practice PVP server for the Bedrock Edition variant of the game. The network reached a total of more than 9400 registered players and a peak of 32 concurrently online players. Due to a lack of developer motivation and interest in the Minecraft Bedrock Edition community itself, as well as the notorious reason of financing, Helios Practice closed official on February 1st, 2021, marking the end of the server's 8-month period, being officially released on June 14th, 2020.
